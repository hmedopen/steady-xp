# Contributing

Thanks for helping improve Steady XP.

Before reporting a bug, verify that it still occurs with the latest release and only
the required loader/API dependencies installed. Include the Minecraft version, loader,
loader version, Steady XP version, relevant logs, and reproduction steps.

Loader-specific code belongs in its matching top-level module (`fabric/`, `neoforge/`,
or `forge/`). Changes that affect every loader should be described as such in the issue
or pull request so they can be applied consistently.
